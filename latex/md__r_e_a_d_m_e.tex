Patro\+Galaxy is a classic space shooter game developed specifically for the \href{https://github.com/BitDogLab/BitDogLab}{\texttt{ Bit\+Dog\+Lab}} embedded systems learning platform. It serves as a practical demonstration of embedded programming techniques using the Raspberry Pi Pico W microcontroller.

See Patro\+Galaxy in action! Watch the project overview video\+:

\href{https://www.youtube.com/watch?v=mwKkf3WDbOc}{\texttt{ }}

\href{https://www.youtube.com/watch?v=mwKkf3WDbOc}{\texttt{ Watch the overview video on You\+Tube}}

This project showcases the capabilities of the Bit\+Dog\+Lab, including its OLED display, analog stick, action button, and flash memory, within the engaging context of a retro-\/style arcade game.\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md__r_e_a_d_m_e_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Classic Space Shooter Gameplay\+:}} Navigate a spaceship, dodge asteroids, and shoot enemies to achieve the highest score.
\item {\bfseries{Bit\+Dog\+Lab Integration\+:}} Leverages the platform\textquotesingle{}s integrated OLED display (SSD1306), analog stick, and action button for a complete interactive experience.
\item {\bfseries{High Score Persistence\+:}} Stores the player\textquotesingle{}s high score in the Raspberry Pi Pico W\textquotesingle{}s flash memory, allowing for persistent record keeping.
\item {\bfseries{Progressive Difficulty\+:}} The game\textquotesingle{}s difficulty increases over time, providing an engaging and challenging experience.
\item {\bfseries{Modular Code Design\+:}} The code is well-\/structured and modular, making it easy to understand, modify, and extend with new features.
\item {\bfseries{Clean Code\+:}} Well documentated for simple understanding
